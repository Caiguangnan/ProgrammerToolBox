#### 2.3 repeat-break循环语句

![image-20200308182733109](C:\Users\tiger\AppData\Roaming\Typora\typora-user-images\image-20200308182733109.png)

```R
#界定pv的级别，返回result字符串向量
pv <- c(1,1,2,3,1,1,15,7,18)
i <- 1
result <- ''
repeat{
    if(i>length(pv)){
        break
    }
    if(pv[i]<=5){
        result[i] <- '初级用户'
    }else if(pv[i]<=15){
        result[i] <- '中级用户'
    }else{
        result[i] <- '高级用户'
    }
       i <- i+1
}
result
```
